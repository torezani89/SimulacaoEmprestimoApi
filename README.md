# SimulacaoEmprestimoApi

API para simulação de empréstimos desenvolvida em ASP.NET Core

## 🚀 Funcionalidades
- Simulação de parcelas de empréstimo
- Cálculo de juros e amortização
- API RESTful

## 🛠️ Tecnologias
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server

dotnet run
